# SKKS Tree Service LP

ห้างหุ้นส่วนจำกัด SKKS ทรีเซอร์วิส — เว็บไซต์นำเสนอบริการรับจ้างตัดต้นไม้แบบเหมา (จังหวัดชลบุรี)

โครงสร้างโปรเจกต์ใช้ [Next.js](https://nextjs.org) (`create-next-app`)

## รันในเครื่อง

```bash
npm install
npm run dev
```

เปิด [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
